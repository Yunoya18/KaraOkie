ประเภทของโปรเจค (ถ้าไม่ครบตามข้อกำหนด คะแนนที่ได้รับจะถูกหัก 50% ขณะที่ จะได้รับ 2 คะแนน ถ้าทำครบตามข้อกำหนด ) ต้องอาศัยความรู้ต่อไปนี้ในการพัฒนาโปรเจค
(1) Multiple Document Interface (MDI) ของบทที่ 8 และ
(2) JTable ของบทที่ 8  หรือ  Thread ของบทที่ 13

โปรเจคต้องได้รับการพัฒนาด้วย (8 คะแนน)
(1) ภาษาจาวา (ถ้าไม่พัฒนาด้วยภาษาจาวา จะได้รับคะแนนโปรเจคเป็น 0 เท่านั้น) และ
(2) เขียนโปรแกรมบนหลักการเชิงวัตถุ
(2.1) ต้องใช้หลักการห่อหุ้ม, สืบทอด และ การมีได้หลากหลายรูปแบบ (override และ overload)
(2.2) ต้องสร้าง (1) Custom Class, (2) Abstract Class และ (3) Interface
(2.3) ต้องสร้างเมธอดที่มีการรับค่าเป็น (1) Custom Class และ (2)  Abstract Class และ (3) Interface (คนละเมธอดกัน)
(2.4) ต้องมีการสร้างและเรียกใช้งาน (1) แอททริบิวท์ของคลาส, (2) แอททริบิวท์ของออปเจ็กต์, (3) แอททริบิวท์ค่าคงที่ของคลาส , (4) แอททริบิวท์ค่าคงที่ของออปเจ็กต์ 
(2.5) มีการสร้าง Custom Class โดยอาศัยหลักการ  Parametric Polymorphism (ในบทที่ 6 หน้าที่ 56 - 58)
(2.6) มีการ Casting Object
ข้อ (2.1) ถึง (2.6) ไม่นับความรู้จากบท GUI และ Event
โปรแกรมสามารถทำงานได้ถูกต้อง ความสมบูรณ์ ความยากง่าย ความน่าสนใจ และ เทคนิคที่ใช้งาน (8 คะแนน)
การนำเสนอและคุณภาพของเอกสารนำเสนอ อาทิเช่น คำผิด ความถูกต้องของคลาสไดอะแกรม ความครบถ้วน ความสอดคล้อง (2 คะแนน)
